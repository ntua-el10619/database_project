## Σύστημα Διαχείρισης Νοσοκομείου "Υγειόπολις"

Στην εργασία αυτή υλοποιήθηκε μια βάση δεδομένων για τη διαχείριση ενός νοσοκομειακού συστήματος. Το σύστημα καλύπτει τη διαχείριση προσωπικού, ασθενών, νοσηλειών, φαρμάκων, εξετάσεων, βαρδιών και άλλων λειτουργιών που απαιτούνται για την οργάνωση ενός νοσοκομειακού περιβάλλοντος.

Κατά την ανάπτυξη έγινε σχεδιασμός ER διαγράμματος και σχεσιακού διαγράμματος και στη συνέχεια υλοποιήθηκε το σχήμα της βάσης σε MariaDB.

## Εκτέλεση

Για την εκτέλεση χρησιμοποιήθηκε MariaDB σε Docker container.

Εγκατάσταση σχήματος:

```bash
docker exec -i ygeiopolis-db mariadb -u root -proot < sql/install.sql
```

Φόρτωση δεδομένων:

```bash
docker exec -i ygeiopolis-db mariadb -u root -proot ygeiopolis < sql/load.sql
```

Παράδειγμα εκτέλεσης ερωτήματος:

```bash
docker exec -i ygeiopolis-db mariadb -u root -proot ygeiopolis < sql/Q01.sql > sql/Q01_out.txt
```

## Δεδομένα που χρησιμοποιήθηκαν

Στη βάση δημιουργήθηκαν και φορτώθηκαν δεδομένα για:

- 95 γιατρούς
- 200 ασθενείς
- 500 νοσηλείες
- 15 τμήματα
- 320 συνταγογραφήσεις
- 150 ιατρικές πράξεις
- 200 εργαστηριακές εξετάσεις

Τα αποτελέσματα των ερωτημάτων αποθηκεύονται στα αντίστοιχα αρχεία `Qx_out.txt`.

Κατά την ανάπτυξη πραγματοποιήθηκαν δοκιμές και έλεγχοι ώστε τα δεδομένα και τα ερωτήματα να επιστρέφουν έγκυρα αποτελέσματα.
