# Σύστημα Διαχείρισης Νοσοκομείου «Υγειόπολις»

## Περιγραφή

Στην παρούσα εργασία υλοποιήθηκε βάση δεδομένων για το νοσοκομειακό
σύστημα «Υγειόπολις» με χρήση MariaDB. Η βάση σχεδιάστηκε ώστε να
υποστηρίζει λειτουργίες σχετικές με τη διαχείριση προσωπικού, ασθενών,
νοσηλειών, εξετάσεων, φαρμάκων, βαρδιών και ιατρικών πράξεων.

Κατά την ανάπτυξη πραγματοποιήθηκε αρχικά σχεδιασμός του ER διαγράμματος
και του σχεσιακού διαγράμματος και στη συνέχεια υλοποιήθηκε το σχήμα της
βάσης.

## Βήματα Υλοποίησης

1.  Σχεδίαση ER διαγράμματος σύμφωνα με τις απαιτήσεις της εκφώνησης.
2.  Δημιουργία σχεσιακού διαγράμματος.
3.  Υλοποίηση του σχήματος μέσω του αρχείου `install.sql`.
4.  Φόρτωση δεδομένων μέσω του `load.sql`.
5.  Εκτέλεση των ερωτημάτων `Q01–Q15`.
6.  Αποθήκευση των αποτελεσμάτων στα αρχεία `Qxx_out.txt`.

## Περιβάλλον Υλοποίησης

-   MariaDB 11
-   Docker
-   Visual Studio Code
-   Draw.io
-   PlantUML
